# 🔐 Secure Login System

## 📌 Project Description

This project is a **Secure Login System Web Application** built using **Python (Flask)** and **SQLite database**.
It allows users to register, login, and access a protected dashboard with secure authentication.

---

## 🚀 Features

* ✅ User Registration
* ✅ Secure Login System
* ✅ Password Hashing using **bcrypt**
* ✅ Session Management (Login/Logout)
* ✅ Protection from SQL Injection
* ✅ Simple and User-Friendly Interface

---

## 🛠️ Technologies Used

* Python
* Flask
* SQLite
* HTML
* bcrypt

## ▶️ How to Run the Project

1. Install required libraries:

```
pip install flask bcrypt
```

2. Run the application:

```
python app.py
```

3. Open browser and go to:

```
http://127.0.0.1:5000/register
```
